Instructions for Input File

To upload a custom set of inputs (e.g. words, phrases), the input file must:

  • have extension: .txt
  • use encoding: UTF-8
  • have one input (e.g. word, phrase) per row

You can download a template here: inputs-template.txt.
Download it, modify it as needed (e.g. insert your custom words, phrases), and upload it again.
NOTE: Inputs will be shuffled to prevent systemic biases during training.
